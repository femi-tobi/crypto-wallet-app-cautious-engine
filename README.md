## Features

- **100+ cryptocurrencies** with real-time prices
- **Interactive 7-day price chart** using `fl_chart`
- **Search & Favorites** (persistent with Hive)
- **Full offline mode** with cached data
- **Swap screen** with token selection & confirmation
- **Explore tab**: Trending, Gainers, Losers
- **Shimmer loading**, error states, retry logic
- **Hot restart safe** — no white screen
- **Dark Web3 UI** with eye toggle for balance

---

## Tech Stack

| Layer | Technology |
|------|------------|
| **State Management** | `Provider` |
| **Networking** | `Dio` + `dio_cache_interceptor_hive_store` |
| **Local Storage** | `Hive` (favorites, cache) |
| **Charts** | `fl_chart` |
| **Image Caching** | `cached_network_image` |
| **Loading UI** | `shimmer` |
